# Link in Bio – Meu Aprendizado em HTML/CSS

Este projeto é uma página estilo “link in bio” criada para demonstrar meu aprendizado em desenvolvimento web.  
A proposta é construir uma página simples, funcional e responsiva usando apenas HTML e CSS, explorando boas práticas de estrutura, semântica e design.

## 🎯 Objetivo do Projeto

Criar uma página pessoal que reúna meus links e redes sociais enquanto pratico:

- Estruturação semântica em HTML
- Estilização com CSS puro
- Organização de componentes visuais
- Responsividade para dispositivos móveis

## 🧠 O que aprendi

- Criar layouts limpos usando flexbox
- Trabalhar com hierarquia visual e tipografia
- Construir botões estilizados e consistentes
- Organizar arquivos e pastas de forma profissional

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
